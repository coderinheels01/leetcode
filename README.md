# LeetCode Solutions

A collection of LeetCode problem solutions in Python.

## Project Structure

```
leetcode/
├── solutions/       # Problem solutions
├── tests/           # Unit tests
├── pyproject.toml   # Project configuration
└── README.md
```

## Usage

```bash
# Create and activate virtual environment
python3 -m venv .venv
source .venv/bin/activate

# Install in development mode
pip install -e .

# Run tests
pytest
```
